## 실행방법

 저희 Project는 **express** 와 **Vue**를 연동하였습니다. 그래서 실행을 해보실 때 유의 사항을 가르쳐 드리겠습니다.



1. project를 받은 후 "npm build" 명령어를 입력 해주십시오.
2. "backend" 디렉토리로 이동하신 후 "npm start" 명령어를 실행 해주십시오.
3. "localhost:3000" 으로 저희의 홈페이지를 확인 해보실 수 있습니다.
